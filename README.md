## 內容
#### 利用UNITY軟體以及KINECT感測器施作出遊戲，透過地圖連接使玩家能遊玩數個關卡，且角色能使用各式各樣的能力，能力分別如下：
#### 1.飛行 2.躲避物品 3.打怪 4.抓取物品 
## 效益
#### 透過遊戲放鬆身心，提升反應能力、增進親友間的感情
## 限制
### 開發：開發者尚未接觸相關技術
### 使用者：器材限制

#### 4.相容性：低
